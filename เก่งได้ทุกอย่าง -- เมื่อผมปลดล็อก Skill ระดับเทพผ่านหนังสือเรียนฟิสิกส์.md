**ความเข้าใจ**
- คิดก่อนค่อยลงมือทำโจทย์
- พยายามตีความให้ได้ว่าคนออกโจทย์กำลังสื่ออะไร
- ดูองค์ประกอบว่าโจทย์ใช้เนื้อหาอะไรบ้าง แล้วก็ดูโจทย์ก่อนหน้าว่ามีอะไรที่ขาดตกบ้าง
- นำแนวคิดก่อนหน้าไปใช้กับศาสตร์อื่น ๆ
- เนื้อหาไหนที่เราพอจะทำได้แล้ว ลองกระโดดข้ามไปอีก step ดู เพื่อดูว่าเราไปไกลได้อีกแค่ไหน
- ไม่ต้องเร่ง พยายามปูพื้นฐานให้แข็ง
**ความจำ**
- มองภาพรวมของเนื้อหาที่จะจำ ( ดูสารบัญ ) ดูเส้นทางว่าเราจะได้เจอเรื่องอะไรบ้างคร่าว ๆ
- ใช้ technique การจำ
	- Memory Palace Technique
	- เครือข่ายใยแมงมุม - พยายามเชื่อมโยงเนื้อหาเข้าด้วยกัน
- พยายามอย่าติดกับ Muscle Memory
- หลังจากอ่านไปแล้วลองพยายามคิดโจทย์เองดูให้มันยากพอสมควร
**เทคนิคเสริม**
- ใช้ Active Recall + Pomodoro + ออกกำลังกาย
- ไม่ว่าเราจะทำอะไรพยายามนึกเป้าหมายให้ออก ไม่ใช่ว่าทำไปก่อนแล้วค่อยคิด
- ใน paragraph อ่านต้นกับท้าย ( ถ้าคนเขียนเขียนมาดี ) สรุปใจความ
- highlight เฉพาะข้อความสำคัญ ๆ ไม่ใช่ทั้งประโยค

และที่สำคัญที่สุดเลยคือ ***"การลงมือทำ"***
